# SPREAD visual brand essentials

## Colors

| Name | Hex | Use |
|---|---|---|
| **SPREAD Orange** | `#FF6F47` | Primary accent: buttons, highlights, headers. Never for body text or large backgrounds. |
| **Signal White** | `#FFFFFF` | Primary background, text on dark |
| **Premium Dark** | `#272727` | Primary text on light backgrounds, dark backgrounds |
| **Neutral Grey** | `#D1D1D1` | Dividers, secondary structure |
| **Light Grey** | `#F5F5F5` | Section backgrounds, subtle surfaces |

**Orange gradient scale (light → dark):**
`#FFF1ED` → `#FFE2DA` → `#FFD4C8` → `#FFC5B5` → `#FEA991` → `#FE8C6C` → `#FF6F47`

**Rules:**
- Orange is always the accent, never the base. Greys support orange, never dominate it.
- **Text on orange backgrounds:** Use Premium Dark (`#272727`) only. Never use white (`#FFFFFF`) on SPREAD Orange — it fails WCAG AA contrast (≈2.9:1 vs. the required 4.5:1). Premium Dark on orange passes at ≈5.8:1.
- **Text on dark backgrounds:** Use white or orange.


## Typography

| Role | Font | Fallback |
|---|---|---|
| **Headlines** | DIN Alternate Bold | Arial Bold |
| **Subheadings** | DIN Pro Bold | Arial Bold |
| **Body** | DIN Pro Regular | Arial |
| **Annotations/Buttons** | DIN Pro Regular | Arial |

**Avoid:** Mixed weights in a single headline. Colored words within headlines. Wide letter-spacing.

## Logo rules

- Logomark (two-arc symbol) + logotype ("SPREAD") stay together — only separate for avatar/favicon use.
- Always **left-aligned**, never centered.
- Minimum size: **126 px** wide (digital).
- On light backgrounds: orange logomark + orange logotype. On dark: orange logomark + white logotype.
- Never stretch, add effects, or place on busy images.

## Formatting rules (quick ref)

- **Language** Follow the AP Style guide for grammar, captilization, punctuation, and language. Exception: Use the oxford comma.
- **"SPREAD"** is always all-caps in running text.
- **US English** default (exception: defense context uses British spelling).
